Base surface shell — white on light, gray-900 on dark, with a subtle border, 14px radius and a short navy-tinted shadow. Everything sits inside a Card.

```jsx
<Card>…</Card>
<Card variant="inverse">Hero / brand panel on navy</Card>
<Card variant="accent">Próxima melhor ação (success-tinted)</Card>
<Card interactive>Clickable / hover-lift</Card>
```

`padding`: `md` (20) default, `lg` (24), `none` (flush). `variant`: `default | inverse | accent`. Render any tag via `as`.
