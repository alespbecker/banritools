Compact status / role / metric label. Fixes the legacy low-contrast pills — every tint pairs a soft background with a dark, AA-safe label.

```jsx
<Badge variant="brand">+2.550 pts</Badge>
<Badge variant="purple" size="md">TIME</Badge>
<Badge variant="success" dot>Preenchido</Badge>
```

Variants: `neutral | brand | success | warning | error | purple | turquoise | solid`. `purple` is the convention for role tags (Admin/TIME). Use `dot` for live-status, `square` for tabular contexts.
